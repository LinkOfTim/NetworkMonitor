views.graphs module¶

class views.graphs.TrafficGraphs¶

Базовые классы: QWidget

Класс для отображения графиков трафика в приложении с переключением между линейным графиком и круговой диаграммой. Графики обновляются на основе данных сетевого анализатора.

toggle_graph()¶

Переключает текущий тип графика между линейным графиком и круговой диаграммой.

update_graphs(analyzer=None)¶

Обновляет графики с учетом текущих данных. Если передан объект analyzer, извлекает из него данные для графиков.

Параметры:

analyzer – объект анализатора, предоставляющий данные о трафике (необязательно)
