controllers.capture_controller module¶

class controllers.capture_controller.CaptureController(interface: str)¶

Базовые классы: QObject

Класс для управления захватом сетевых пакетов. Захватывает пакеты с указанного сетевого интерфейса в отдельном потоке и периодически отправляет их в виде сигнала.

emit_packets()¶

Извлекает захваченные пакеты из очереди и отправляет их через сигнал packets_received.

packets_received¶

int = …, arguments: Sequence = …) -> PYQT_SIGNAL

types is normally a sequence of individual types. Each type is either a type object or a string that is the name of a C++ type. Alternatively each type could itself be a sequence of types each describing a different overloaded signal. name is the optional C++ name of the signal. If it is not specified then the name of the class attribute that is bound to the signal is used. revision is the optional revision of the signal that is exported to QML. If it is not specified then 0 is used. arguments is the optional sequence of the names of the signal’s arguments.

Type:

pyqtSignal(*types, name

str = …, revision

controllers.filter_controller module¶

class controllers.filter_controller.FilterController¶

Базовые классы: object

Класс для управления фильтрами сетевых пакетов. Позволяет применять фильтры по протоколу, IP-адресу отправителя и IP-адресу получателя.

apply_filters(packet)¶

Проверяет, проходит ли пакет через фильтры.

Параметры:

packet (NetworkPacket) – Захваченный пакет.

Результат:

True, если пакет проходит фильтры.

Тип результата:

bool

class controllers.save_controller.SaveController(parent)¶
